Guides

Core Concepts

Nova is built on the philosophy of "Zero-Gate" development. We remove the middleman between you and your music metadata.

INFO Zero Configuration

Nova runs locally or in a private container. Default access is anonymous—no API keys or headers required in development mode.

Search API

Powerful multi-source search for tracks, artists, and albums with deep formatting.

Tracks & Streaming

The core of Nova. High-fidelity audio streaming with metadata parity.

STREAM /api/v1/tracks/stream/{id}

Returns an optimized audio/webm or audio/m4a stream directly from the engine.

GET /api/v1/tracks/{id}

Returns detailed metadata, including high-res thumbnails and track duration.

Browse & Moods

Explore charts, trending songs, and categorized genre playlists.

GET /api/v1/browse/trending?region=US
GET /api/v1/browse/genres

Python SDK

The most elegant way to interact with Nova. Zero-config by default.

# Installation: pip install nova-music-sdk
from nova import NovaClient

# Connect to local core
client = NovaClient()

# Integrated search & stream
results = client.search("Solana")
track_url = client.get_stream_url(results['data']['tracks'][0]['id'])

Error Codes

Nova uses structured JSON error responses with unique codes for debugging.

Code Status Context
TRACK_ERROR 500 Upstream source unreachable.
MISSING_QUERY 400 Search requires the 'q' parameter.
FORBIDDEN 403 IP blocked or API Key invalid.
NOT_FOUND 404 The requested entity was not found.
